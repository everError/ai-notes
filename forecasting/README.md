# Time Series Forecasting

시계열(Time Series) 데이터의 패턴을 분석하고 미래 값을 예측하는 모델과 기법을 학습하고 정리합니다.

주가, 매출, 트래픽, 센서 데이터, 전력 사용량 등 시간의 흐름에 따라 수집되는 데이터를 대상으로 예측(Forecasting)을 수행하는 방법을 다룹니다.

---

## Topics

### Foundation Models

범용 시계열 예측 모델을 학습합니다.

- [TimesFM](./TimesFM/)
- Chronos
- Moirai

---

### Deep Learning

딥러닝 기반 시계열 예측 모델을 학습합니다.

- LSTM
- GRU
- Transformer
- PatchTST

---

### Statistical Models

전통적인 통계 기반 시계열 예측 모델을 학습합니다.

- AR
- MA
- ARIMA
- SARIMA
- Prophet

## Notes

시계열 예측 분야는 전통적인 통계 모델에서 시작하여 딥러닝, Transformer 기반 모델을 거쳐 최근에는 Foundation Model 중심으로 발전하고 있습니다.

이 디렉터리에서는 각 모델의 개념, 구조, 사용 방법 및 실습 내용을 정리합니다.
